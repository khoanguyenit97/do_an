Ứng dụng hỗ trợ các ký hiệu sau để nhập một biểu thức logic:
  • Phép phủ định (¬): ~
  • Phép hội (∧): &
  • Phép tuyển (∨): |
  • Phép kéo theo (→): =>
Ví dụ: Để nhập biểu thức ((q → r) ∧ (q ∨ ¬p)) → (p → r) thì sử dụng các ký hiệu như sau ((q => r) & (q | ~p)) => (p => r)
Hai biểu thức cần so sánh được đặt trong cặp ngoặc nhọn {} và cách nhau bởi dấu phẩy (,)
Ví dụ: để so sánh hai biểu thức (p → (q → (p ∧ q ∧ r))) và q → (p → r) , nhập: {(p => (q => (p & q & r))), q => (p => r)}
